# Clon E-commerce Tech

Ejercicio de creación de una API Rest con Spring Boot en Java para emplearla en una website de ventas de productos en linea, para el uso del sitio web debera descargar todo el zip y ejecutar primeramente el servidor local con springboot (la API), para posteriormente abrir el archivo `index.html` y usar el aplicativo web.
